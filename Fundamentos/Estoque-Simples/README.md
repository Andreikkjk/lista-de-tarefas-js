# 🏪 Estoque PRO | Premium Inventory System

Um sistema de gerenciamento de estoque moderno, elegante e funcional, desenvolvido com foco em UI/UX de alta qualidade e performance.

![Preview do Projeto](https://img.shields.io/badge/UI-Glassmorphism-7c3aed?style=for-the-badge)
![Tecnologias](https://img.shields.io/badge/Stack-HTML5%20|%20CSS3%20|%20JS-0ea5e9?style=for-the-badge)

## ✨ Funcionalidades

- **Dashboard Inteligente:** Visualize o total de itens e produtos com estoque baixo instantaneamente.
- **Filtros Dinâmicos:** Pesquisa em tempo real por categoria e faixa de preço.
- **Gestão Simplificada:**
    - Venda de itens com um clique.
    - Reposição de estoque via prompt interativo.
    - Cadastro de novos produtos através de um modal elegante.
- **Terminal de Sistema:** Log em tempo real de todas as ações realizadas (vendas, reposições, cadastros).
- **Design Premium:** Interface baseada em *Glassmorphism*, com animações suaves e ícones modernos.

## 🚀 Tecnologias Utilizadas

- **HTML5:** Estrutura semântica.
- **CSS3 (Vanilla):** Sistema de design customizado, variáveis CSS, animações e efeitos de vidro.
- **JavaScript (ES6+):** Lógica de manipulação de dados, filtros e gerenciamento de estado da UI.
- **Phosphor Icons:** Biblioteca de ícones versátil e moderna.
- **Google Fonts:** Tipografia selecionada (Inter & Outfit) para máxima legibilidade.

## 🛠️ Como Utilizar

1. Faça o download ou clone o repositório.
2. Abra o arquivo `index.html` em qualquer navegador moderno.
3. Utilize o painel lateral para filtrar ou adicionar novos produtos.
4. Gerencie seu estoque diretamente na tabela de inventário.
