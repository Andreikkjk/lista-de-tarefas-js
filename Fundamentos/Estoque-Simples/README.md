# 🏪 Estoque PRO | Premium Inventory System

A modern, elegant, and functional inventory management system developed with a focus on high-quality UI/UX and performance.

![Project Preview](https://img.shields.io/badge/UI-Glassmorphism-7c3aed?style=for-the-badge)
![Technologies](https://img.shields.io/badge/Stack-HTML5%20|%20CSS3%20|%20JS-0ea5e9?style=for-the-badge)

## ✨ Features

- **Smart Dashboard:** Instantly view total items and products with low stock.
- **Dynamic Filters:** Real-time search by category and price range.
- **Simplified Management:**
    - One-click item sales.
    - Stock replenishment via interactive prompt.
    - New product registration through an elegant modal.
- **System Terminal:** Real-time log of all actions performed (sales, replenishments, registrations).
- **Premium Design:** Glassmorphism-based interface with smooth animations and modern icons.

## 🚀 Technologies Used

- **HTML5:** Semantic structure.
- **CSS3 (Vanilla):** Custom design system, CSS variables, animations, and glass effects.
- **JavaScript (ES6+):** Data manipulation logic, filters, and UI state management.
- **Phosphor Icons:** Versatile and modern icon library.
- **Google Fonts:** Selected typography (Inter & Outfit) for maximum readability.

## 🛠️ How to Use

1. Download or clone the repository.
2. Open the `index.html` file in any modern browser.
3. Use the sidebar to filter or add new products.
4. Manage your stock directly in the inventory table.
